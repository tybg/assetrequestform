MULLEN ASSET REQUEST FORM

OVERVIEW
• Complete ALL relevant information to expedite.
• Asset management handles ONLY inquiries regarding digital asset rights, file requests, and the release / delivery of files. Please contact relevant production department for any rights that may need to be purchased, e.g., Art Production for the purchase of image rights.
• Asset management will not deliver a file to print production if usage is not cleared or creatively approved.
• Include file number and image reference of all files requested.
• Standard requests are for FLATTENED files only. Requests for layered files are subject to approval from Art Production. Only final artwork will be released to external vendors.
• Include the Job # for any billable costs.
• Average requests need 2-5 days processing time.
• RUSH is 1-2 days processing (no guarantee).

TIME FACTORS
• Accuracy of file information
• Number of files
• Volume of requests that Asset Manager receives at any given time
• If images need to pass through Art Production for additional research
• Fee pricing

TURNAROUND TIME
Please inform end users of the following turnaround times:
• Up to 3 days for 1-5 images
• Up to 5 days for 6-20 images
• For large requests of 20 or more images, the time frame will be determined on a case by case basis
• RUSH is next day and cannot always be guaranteed
